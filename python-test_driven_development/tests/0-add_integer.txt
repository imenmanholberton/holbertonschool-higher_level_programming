Now use it:
=========
``add_integer()``` returns the addition of two argument.
::
    >>> add_integer = __impor__(`0-add_integer`).add_integer
    >>>add_integer(5, 6)
    11
::
    >>> add_integer(5, -6)
    -1
The function also works with floating.
::
    >>> add_integer(5.0, 6.0)
    11
Floating ana non-floating point value can be combined.
::
    >>> add_integer(2.5, -3)
    -1
The second argument is optional -by default, it is 98.
::
    >>> add_integer(1)
    99
TypeError
==========
``add_integer`` expects that both arguments are either integers or floats.
if either argument is a non_integer and non-float, a TypeError is raised:
::
    >>> add_integer("Bonjour", 3)
    TypeError: a must be an integer
::
    >>> add_integer(3, "Bonjour")
    TypeError: b must be an integer
